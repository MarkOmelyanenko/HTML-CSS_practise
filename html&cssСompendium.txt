<a href="" target=""></a> {
ссылки, в href адрес ссылки, в target указываем где открыть ссылку(новая вкладка, новое окно. 
"_blank" открывает в новой вкладке)
}

<h1></h1> {
    заголовки, таких заголовков считаем до <h6></h6>
}

<strong> {
    делает текст жирным, типа bold
}

<em></em> {
    делает текст курсивным
}

<small></small> {
    делает текст большим(юзаем для слов)
}

<big></big> {
    делает текст маленьким(юзаем для слов)
}
<br> {
    делает переход на следующую строку,
    ставим после строки, после которой хотим поставить enter
    пример: 
    <strong>Это strong текст!</strong><br>
    <em>Это em текст!</em>                 "Это em текст!" будет в некст строке
}

<p></p> {
    делает параграф
}

<ol>

    <li>Элемент</li>
    <li>Элемент</li>
    <li>Элемент</li>
    <li>Элемент</li>

    Это ОRDERED LIST, слева у него цифры
}

<ul>
    <li>Элемент</li>
    <li>Элемент</li>
    <li>Элемент</li>
    <li>Элемент</li>
    <li>Элемент</li>

<img src="" alt="" width="" height=""> {
    что бы вставить фото. если фото не иделально квадратное,
    то можна указать или width, или height и оно само идеально форматирует картинку
}

<div></div> {
    для объединения элементов в один блок
}

<table  border="5">
<th></th>
<tr></tr>
<td></td>
</table> {
    таблица,
    border="" для указания обводки
    <tr></tr> для указания строк, <td></td> для указания столбцов,
    <th></th> для указания главных строк? ну короче, для указания тех, что сверху
}

<input value="Введите что-то">
<input placeholder="Что будем искать?"> {
    поле для ввода
    value="" в поле уже введена какая-то надпись, которую можно стереть
    placeholder="" отображает полупрозрачную надпись в поле, нельзя стереть
}

<textarea cols="50" rows="10"></textarea> {
    используем для ввода больших текстов
    cols="" выставляем значение количества символов в одной строке
    rows="" выставляем значение количества строк
}

<audio controls>
    <source src="audio/04-Ravel.Bolero(extract).mp3" type="audio/mpeg"> 
</audio> {
    controls указываем для отображения кнопок плеера
    <source src="" type="audio/mpeg"> указываем путь и тип
}

<video controls>
    <source src="video/videoplayback.mp4"> 
</video> {
    controls указываем для отображения кнопок плеера
    <source src="" type="video/mpeg"> указываем путь и тип,
    но чёт не робит
}



----------------------------------------CSS------------------------------------------

a { /*селектор по имени тега*/
    color: red;
    font-size: 20px;
}

#my_link { /*значение по идентификатору*/
    color: lime;
    font-size: 25px;
}

.animal { /*значение по классу*/
    font-size: 50px;
}

.message:hover { /*это событие "сделать изменения при наведении"*/
    border: 5px solid rgb(255, 33, 33);
}

-----СВОЙСТВА-----
1. color цвет текста
2. font-size размер шрифта
3. background-color цвет фона
4. border: 1px solid/dashed green; это для рамок: размер рамки | линия/обводка | цвет
5. margin внешние отступы
6. padding внутринние отступы
7. background-image фото на фон
8. border-radius радиус рамки
9. opacity - прозрачность(1 - полсностью видим,
0.5 - на половину прозрачный)
10. box-shadow(принмиает 4 значения: на сколько двигаем по ОХ,
на сколько двигаем по ОУ, парметр размытия тени, цвет)
11. width задет ширину
12. height задает высоту
13. display показывать элемент или не показывать(стандартное значение block)
14. text-align выравнивание шрифта
15. font-weight жирность шрифта(указываем без px, стандартное значение 400, 
900 самый жирный)

/*в transform'е можно перечислять эти функции через пробел*/

16. transform: rotate(90deg) поворачивает на 90 градусов
17. transform: scale(0.5) увеличивает/уменьшает, стандатрное значние 1.0
18. transition: время через сколько элемент должен что-то сделать(1.0 = 1 секунда), далее указываем 
что будет анимировать. можно так задавать: transition: 5s all;
